Website Evaluation - Nicole Cahoon, CPA

I've decided to analyze the website of Nicole Cahoon, a local accountant. During my exploration of her website, I was able to determine 3 major aspects which could use improvement.

Layout

There are a few layout issues with the site which make it feel unprofessional. This is (likely) antithetical to the way Nicole wants to portray her brand.

Headers

For every page on the website, the same header image is used. This header image does not accurately reflect the content of the various pages its used on. Even worse, using the same image over and over comes across as lazy.

Using the dog picture on the home page is fine, but for the other pages the designer consider using more relevant pictures. When talking about how you can help local businesses, why not use a picture of a client's storefront, or a wide shot of downtown Courtenay?

Additionally, the header image used is set to display far too large. On my monitor, which is a standard size of 1920x1080p, the header images take up the entire viewport and the header text itself doesn't show up until you scroll halfway down the page. On mobile, these headers render more reasonably, indicating that perhaps the website was designed with a mobile-first ideology.

Miscellaneous/Smaller Issues

  • Awkward whitespace everywhere, like in the footer, or on the Contact and BC Financial Resources pages.
  • The Facebook button in the nav bar feels unnecessary and out of place, being an off-site link that looks exactly like and is in the same place as the on-site navigation.

Colour

The 3 main colours used, pure black, pure white, and pure yellow, are piercing and hard on the eyes. There are insufficient contrast ratios in many places, namely the testimonials page and home page, where combinations of grey and black are used.

This is easily fixable, and largely up to personal taste. If I were redesigning this page, I might use an off-white background and off-black text, which would be easier on the eyes. Yellow can still work as an accent colour, but I would definitely tone it down.
